课程智慧教学平台

课程智慧教学平台具有管理员、教师、学生三种用户角色，具有用户管理、课程管理、班级管理、作业管理（辅以AI助教）、题型和题库管理、考试管理等必做功能，以及课程知识图谱和实验在线评测等相关智能功能（二选一选做）。

一、用户管理系统（RBAC 模型）

本模块基于角色的访问控制（RBAC）设计，区分以下三种角色及其权限范围：

### 1. 系统管理员 (Admin)
**职责**：平台的整体运维与教师账户管理。
- **教师管理**：
  - 创建教师账户（分发账号给教师）。
  - 查看/禁用/重置教师账户密码。
- **系统监控**：查看平台整体注册人数、活跃度等概览数据。

### 2. 教师 (Teacher)
**职责**：课程教学、班级管理与学生账号维护。
- **班级与学生管理**：
  - **班级创建**：为每门课程创建教学班级。
  - **学生导入**：支持 Excel/CSV 批量导入学生名单（包含学号、姓名）。
  - **账号生成**：导入名单时自动为学生生成账号（默认密码机制）。
  - **账号维护**：重置所辖班级学生的密码、修改学生基本信息。
- **课程与教学**：发布实验、管理知识图谱、批改作业。

### 3. 学生 (Student)
**职责**：完成学习任务。
- **个人中心**：修改个人密码、绑定邮箱/手机号。
- **学习概览**：查看所属课程、待办实验及成绩统计。

二、课程管理（服务教师、学生角色）

下面是该模块主要功能，请各小组基于此进行扩展设计和实现：
核心页面：课程列表页、课程详情页、课程创建 / 编辑页、课程状态管理页。
功能细节：
列表页支持按课程名搜索，可删除；
详情页展示课程大纲、授课教师、关联班级、历史作业 / 考试记录；
创建 / 编辑页包含表单（课程信息），支持富文本编辑课程介绍。

三、班级和学生管理（服务教师角色）

下面是该模块主要功能，请各小组基于此进行扩展设计和实现：
核心页面：班级列表页、班级详情页、班级创建页。
功能细节：
列表页按年级、按照班级分组展示，显示班级名称、班主任、学生人数、成立时间；
详情页包含班级基本信息、学生名单、关联课程列表；
创建页支持选择年级 / 专业、填写班级名称、关联教师账号；
学生管理支持手动添加（输入学号关联学生账号）、移除学生。

四、作业管理（服务教师、学生角色）

下面是该模块主要功能，请各小组基于此进行扩展设计和实现：
核心页面：作业列表页、作业创建页、作业详情页。
功能细节：
列表页按课程、班级、提交状态（未开始 / 进行中 / 已截止）筛选，显示作业名称、截止时间、已提交 / 总人数；
创建页支持选择关联课程 / 班级、设置截止时间、填写作业要求；
批改页可查看单学生提交的作业，支持打分（0-100 分）、填写评语。

五、课程AI助教功能（服务学生角色）

下面是该模块主要功能，请各小组基于此进行扩展设计和实现：
核心页面：AI会话页面、AI会话历史页面、AI会话菜单
会话菜单页面：能够根据作业题目自动弹出AI解析

六、题型和题库管理（服务教师角色）

下面是该模块主要功能，请各小组基于此进行扩展设计和实现：
核心页面：课程题型列表页，题型下属题库列表页、题目添加 / 编辑页、题目批量操作页。
功能细节：
题目添加支持选择题型，录入题干、选项（客观题）、答案、解析，标记难度；

七、考试管理（服务教师、学生角色）

下面是该模块主要功能，请各小组基于此进行扩展设计和实现：
核心页面：课程下的考试列表页、考试创建页、考试详情页，考试结果展示分析页。
功能细节：
列表页按课程、状态（未开始 / 进行中 / 已结束）筛选，显示考试名称、时长、开始时间、关联班级；
创建页支持设置考试名称、时长、开始 / 结束时间、选择关联课程，可关联题库中的题型与内容；

八、课程知识图谱功能（服务教师、学生角色）

下面是该模块主要功能，请各小组基于此进行扩展设计和实现：
核心页面：课程知识图谱列表，按章显示知识点列表，章节/知识点编辑、详情和搜索等页面
关联页面：在实验、作业、题目等编辑页面能关联课程知识点、在作业（实验）查看页面能查看该项作业（实验所涉及的知识点）

九、实验和执行评测管理（服务教师、学生角色）

下面是该模块主要功能，请各小组基于此进行扩展设计和实现：
核心页面：实验列表页、实验创建页、实验详情页、实验回答输入页、实验执行页、实验评测结果页。
功能细节：
实验回答输入页支持学生输入代码；
实验评测结果页显示调用编译环境执行编译之后的结果页。
